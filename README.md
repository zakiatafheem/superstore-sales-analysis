# 📊 Superstore Sales Analysis & Automated Reporting Dashboard

## 📌 Project Overview
This project analyzes retail sales data to uncover key business insights related to sales performance, profitability, and customer trends. It combines Python-based data analysis with an interactive Power BI dashboard and automated reporting.

---

## 🔧 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- Excel
- Jupyter Notebook

---

## 📊 Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization using Python
- Automated report generation using Python script (`report.py`)
- Interactive Power BI dashboard with filters and KPIs

---

## 📈 Analysis Performed
- Monthly Sales Trend Analysis (Line Chart)
- Category-wise Sales Analysis (Bar Chart)
- Sub-Category Sales Analysis (Horizontal Bar Chart)
- Profit Distribution (Box Plot)
- Discount vs Profit Relationship (Scatter Plot)

---

## 📊 Dashboard Highlights (Power BI)
- KPI Cards: Total Sales, Total Profit, Total Orders, Average Discount
- Interactive Filters: Region, Category, Segment
- Sales trend visualization
- Profit and discount relationship analysis
- Identification of high-performing and loss-making categories

---

## 🤖 Automation
Developed a Python script to:
- Clean and preprocess raw data
- Generate insights automatically
- Create visualizations (saved as images)
- Export structured Excel reports
